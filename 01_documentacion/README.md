# Documentacion

Esta carpeta agrupa toda la documentacion tecnica y de apoyo del proyecto.

## Partes
- `manuales/`: hojas tecnicas y manuales de componentes.
- `resumenes/`: documentos de concepto y alcance del sistema.
- `bocetos/`: ideas iniciales, bosquejos y notas visuales.
- `diagramas/`: diagramas de bloques, flujo, conexionado y arquitectura.

Mantener aqui informacion de referencia para diseno, montaje y validacion.
