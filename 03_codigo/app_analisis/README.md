# App de Analisis

Herramientas para analizar la dinamica del golpe y generar resultados.

## Objetivo
- Procesar datos de aceleracion/fuerza.
- Calcular metricas (pico, impulso, duracion, energia estimada).
- Visualizar series temporales y comparativas.

Aqui pueden ir scripts en Python, MATLAB, o aplicaciones de escritorio/web.
