# Librerias Externas

Dependencias de terceros utilizadas por firmware y herramientas de analisis.

## Recomendaciones
- Separar por tecnologia (por ejemplo: `stm32/`, `python/`, `web/`).
- Guardar version y origen de cada libreria.
- Evitar modificar librerias sin documentar los cambios.

Incluir notas de licencia cuando corresponda.
