# Datos de Pruebas

Resultados de ensayos y mediciones del sistema.

## Contenido sugerido
- Registros crudos de sensores.
- Datos ya procesados.
- Metadatos de prueba (fecha, configuracion, sujeto/dispositivo).
- Graficas y reportes.

Organizar por fecha o por tipo de experimento para facilitar comparaciones.
