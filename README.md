# P H-it

Proyecto para medir la intensidad y dinamica de golpes (por ejemplo, un punetazo), integrando electronica, firmware y analisis de datos.

## Estructura general
- `01_documentacion/`: manuales, resumenes, bocetos y diagramas.
- `02_diseno/`: plantillas visuales, mockups y recursos de presentacion.
- `03_codigo/`: codigo fuente del sistema (firmware, app de analisis, utilidades).
- `04_librerias_externas/`: dependencias de terceros, SDKs y librerias.
- `05_datos_pruebas/`: resultados de mediciones, capturas y datasets de prueba.

## Siguiente paso recomendado
Completar `03_codigo/` con los primeros modulos de adquisicion de senal y procesamiento del golpe.
